# Methodologies Quick Reference
## One-Page Synthesis: When to Use What

Each methodology is a lens. The same script looks different through each one. Use the right lens for the right problem — then stop and write.

---

## Framework Summaries

### Peter Russell — BMOC (Scene Level)
**Core tool:** Beat Card (15 fields) + 6 rewrite passes  
**Key question:** What is the binary beat question? Who is the hero of this beat?  
**Signature concept:** Every scene is a mini-war. It must turn and sting.  
**Use for:** Flat scenes, missing tension, passive scenes, mechanical dialogue  
**Read:** `references/scene-engineering.md`

---

### K.M. Weiland — Character Arc + Structure
**Core tool:** Ghost → Lie → Want → Need → Theme pipeline + structural percentage map  
**Key question:** What false belief does this character hold, and how does the structure force them to confront it?  
**Signature concept:** Structure IS the externalized shape of a character's internal journey.  
**Use for:** Weak character arcs, disconnected plot/character, unearned transformation, flat midpoints  
**Read:** `references/character-depth.md` (pipeline) and `references/macro-structure.md` (structure map)

---

### Jeff Lyons — Rapid Story Development (Moral Component + Narrative Engine)
**Core tool:** Moral Blind Spot + Immoral Effect + Offer-Refusal Loop + Pattern of Decline/Elevation  
**Key question:** Does the protagonist CAUSE their own problems through the action of their blind spot?  
**Signature concept:** If you remove the protagonist from the middle, do events still happen the same way? If yes, they're passive.  
**Use for:** Mushy middles, passive protagonists, weak opponents, missing thematic engine  
**Read:** `references/character-depth.md` (moral component + opponent) and `references/macro-structure.md` (narrative engine)

---

### Enneagram Psychology
**Core tool:** 9-type system — core fear, desire, survival strategy, blind spot, evolution/de-evolution points  
**Key question:** Which type's core fear is at the center of this story? Does the opponent attack it specifically?  
**Signature concept:** The de-evolution point is the opponent's type. The evolution point is the Need made flesh.  
**Use for:** Shallow character psychology, opponent that feels generic, arc direction unclear  
**Read:** `references/character-depth.md` (Enneagram integration section)  
**Full tools:** `enneagram-analyst` (typing existing scripts) / `enneagram-architect` (building from scratch)

---

### Story Grid — Genre and Five Commandments
**Core tool:** External genre (obligatory scenes + conventions) + Five Commandments (Inciting Incident, Turning Point, Crisis, Climax, Resolution)  
**Key question:** Is the genre promise being honored? Are all obligatory scenes present?  
**Signature concept:** Genre is a contract. Breaking it loses audiences. Honoring it — then surprising within it — earns them.  
**Use for:** Story that doesn't deliver emotionally, missing genre beats, unclear what type of story this is  
**Read:** `references/genre-system.md` and `references/macro-structure.md`

---

### Save the Cat (Blake Snyder)
**Core tool:** 15 beat sheet across 110 pages  
**Key moments:** Opening image, catalyst (p12), debate, break into 2 (p25), B story (p30), midpoint (p55), all is lost (p75), dark night (p85), finale (p85–110), final image  
**Key question:** Does the opening image answer directly to the final image? Has the world fundamentally changed?  
**Signature concept:** B story (the love/mentor story) carries the theme while A story carries the plot.  
**Use for:** Commercial feature structure, page-count calibration, logline development  
**When to use:** Feature films with strong commercial genre. Less useful for prestige/arthouse tone.

---

### Syd Field — The Paradigm
**Core tool:** Three-act structure with two plot points as hinges (pp25-27, pp85-90)  
**Key question:** Does Plot Point 1 definitively spin the story in a new direction? Does Plot Point 2 set up the climax?  
**When to use:** Classic three-act diagnosis. Good for identifying if the act turns are actually turning.

---

### John Truby — 22 Steps + Genre Forms
**Core tool:** 22-step character-driven structure + detailed genre form analysis  
**Key concepts:** Moral argument, opponent as character's mirror, 7 steps of story structure  
**Signature concept:** The protagonist and antagonist are the same person — the opponent is who the hero would be without self-awareness.  
**Use for:** Deep character-opponent relationship, genre form specificity, moral argument clarity  
**Read:** `references/genre-system.md` (Truby genre networks section)

---

### Hero's Journey (Campbell / Vogler)
**Core tool:** 12-stage mythic structure (Ordinary World → Call → Threshold → Ordeal → Reward → Return)  
**When to use:** Myth-scale stories, genre films with archetypal dimensions, diagnosing if a story lacks mythic resonance  
**Limitation:** Generic at the beat level. Works better as diagnostic than prescriptive.

---

### Vonnegut Story Shapes
**Core tool:** Emotional value charted on a timeline — the "Story of a Story"  
**8 curves:** Man in Hole, Boy Meets Girl, Cinderella, Old Testament Fall, New Testament, Hamlet (flat then fall), Kafka (flat and bad), etc.  
**When to use:** Quickly identifying the emotional arc's shape. Useful for diagnosing if the emotional rhythm is working.

## Methodology Combinations That Work

| Problem | Combination |
|---------|-------------|
| Flat story with thin characters | Lyons (moral component) + Enneagram (type psychology) |
| Good characters, broken structure | Weiland (arc map) + Story Grid (Five Commandments) |
| Structure works, scenes are flat | BMOC (scene engineering) + Dialogue craft (tactic catalog) |
| Everything works, nothing special | Originality Engine (elevation moves) |
| Genre doesn't deliver | Story Grid genre obligations + Truby genre forms |
| Protagonist feels passive | Lyons offer-refusal loop + BMOC hero-of-the-beat discipline |
| Arc feels mechanical | Weiland Ghost-Lie-Want-Need + Enneagram evolution/de-evolution |

## Rule of One Problem

Identify the **single deepest problem** before applying any methodology. The deepest problem is usually:
- Missing moral component → everything else is downstream of this
- Unclear Lie → arc can't work without it
- Passive protagonist → story doesn't have an active engine

Fix the deepest problem first. Secondary problems often resolve themselves.

**Do NOT:** Run all methodologies simultaneously. The signal gets lost in the noise. Pick the lens that illuminates the problem, apply it, then check whether other problems remain.
