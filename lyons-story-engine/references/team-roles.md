# Team Roles Across Four Stages

Source: Jeff Lyons, Rapid Story Development (Routledge, 2020).

Use this to give every character in a team story a specific dramatic job at each stage, not just "plucky comic relief" or "the skeptic." Each type behaves differently depending on whether the team is forming, storming, norming, or performing.

## Six Team Story Characteristics

A team story must have all six:
1. A clear protagonist carries the story
2. Single main goal/task
3. Plot is time sensitive
4. Subplots serve the mainline story
5. Story develops in stages
6. Continuous narrative, not episodic

## Four Stages of Team Development

- **Forming**: Team comes together. Roles clarify. Honeymoon.
- **Storming**: Internal conflicts. Power struggles. Leadership challenged.
- **Norming**: Working agreements. Cohesion begins. Common ground.
- **Performing**: Peak operation. Unified against opponent.

## 9-Type x 4-Stage Behavior Matrix

### Type 1 -- The Reformer
| Stage | Behavior |
|-------|----------|
| **Forming** | Task focused, minimal need for social connection; may suggest ways to structure the work |
| **Storming** | Frame conflict as problem to solve, exert leadership, grow impatient if conflict persists |
| **Norming** | Suggest rules for working together |
| **Performing** | Embrace high production of excellent quality |

### Type 2 -- The Helper
| Stage | Behavior |
|-------|----------|
| **Forming** | Encourage contributions; facilitate organizing around central purpose |
| **Storming** | Assist others to express feelings toward resolution; may give advice or distract |
| **Norming** | Push team for clear, shared working agreements |
| **Performing** | Like performing, particularly in support of star performers |

### Type 3 -- The Achiever
| Stage | Behavior |
|-------|----------|
| **Forming** | Seek team approval early; may assert to define goals |
| **Storming** | Become disengaged; perceive conflict as wasting time and too emotional |
| **Norming** | Like unification, refocus on results and efficient work |
| **Performing** | Favorite stage; encourage others to increase performance |

### Type 4 -- The Individualist
| Stage | Behavior |
|-------|----------|
| **Forming** | Focus on own internal feelings more than task |
| **Storming** | Enjoy realness of explicit conflict as long as they are not main participant |
| **Norming** | Suggest/support rules that add clarity but not restrict individuality |
| **Performing** | Feel more part of the team; work hard toward common goals |

### Type 5 -- The Investigator
| Stage | Behavior |
|-------|----------|
| **Forming** | Social connections feel frivolous yet necessary; strong preference for focus on task |
| **Storming** | Prefer to skip this stage to avoid anger and conflict |
| **Norming** | Like clear rules/structure if sufficient autonomy allowed |
| **Performing** | Prefer individual tasks in areas of competence; like being appreciated for knowledge |

### Type 6 -- The Loyalist
| Stage | Behavior |
|-------|----------|
| **Forming** | Prefer watching dynamics; will clarify issues or ensure vulnerable members get heard |
| **Storming** | May engage if authority/power issues involved; may also withdraw |
| **Norming** | Work actively to secure agreement; attention to equal participation |
| **Performing** | Keep others focused on deliverables; troubleshoot; acknowledge contributions |

### Type 7 -- The Enthusiast
| Stage | Behavior |
|-------|----------|
| **Forming** | Contribute ideas about larger vision; dislike too much structure; impatient with process |
| **Storming** | Dislike conflict; see disagreements as trivial; use humor to defuse |
| **Norming** | Suggest minimal rules that include everyone; react against limiting rules |
| **Performing** | Work from own prioritizations; prefer variety and positive social interaction |

### Type 8 -- The Challenger
| Stage | Behavior |
|-------|----------|
| **Forming** | Either suggest direction or pull back and watch, deciding whether to be part of the group |
| **Storming** | Enjoy intensity if people are honest; often part of conflict; if not, will lead resolution |
| **Norming** | Recommend ground rules, especially allowing everyone to be heard |
| **Performing** | If work is exciting, they stay; if not significant or too predictable, they move on |

### Type 9 -- The Peacemaker
| Stage | Behavior |
|-------|----------|
| **Forming** | Difficulty focusing if progress slow; may become impatient |
| **Storming** | Uncomfortable and looking for a way out |
| **Norming** | Ambivalent about agreements; like consensus but don't like arbitrary rules |
| **Performing** | Getting something accomplished with harmonious team very pleasurable |
