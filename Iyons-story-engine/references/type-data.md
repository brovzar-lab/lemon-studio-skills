Complete narrative profiles for all nine types. Source: Jeff Lyons, Rapid Story Development (Routledge, 2020).

When working with a protagonist or opponent, read the FULL profile for their type. Don't summarize or generalize. Cite specific fields.

## Type 1 -- The Reformer

| Field | Data |
|-------|------|
| **Center** | Body (Anger/Resentment) |
| **What We Like** | Eye for detail, keep word, responsible, diligent, principled, discerning |
| **What We Don't Like** | Judgmental, resentful, controlling, hyper-critical, perfectionist, rigid |
| **Key Motivations** | Do it right, self-justification, be beyond criticism, have integrity |
| **Core Fear** | Change, chaos, paradox |
| **Poison** | Resentment leading to self-loathing, feeling defective |
| **Core Desire** | Integrity |
| **Distortion** | Critical perfectionism, judgmental intolerance |
| **Focus of Attention** | What is right or wrong, correct or incorrect? |
| **Survival Strategy** | Follow the rules to get approval of the one with power |
| **Core Image** | I am reasonable |
| **Parental Matrix** | Negative with father |
| **Speaking Style** | Precise, direct, exacting, concise, detailed. Use should/ought/must/correct/right |
| **Speaking Attitude** | Self-constrained, unblemished, directive, inflexible, confident, stolid |
| **Communication Blind Spots** | Appear critical, impatient, angry. Tenacious about own opinions. |
| **Distortion Filters** | Being criticized; preoccupied with own ideas; whether others behave correctly |
| **Common Pinches** | Being criticized; lack of follow-through; non-collaborative plan changes; deception |
| **Pinch Behaviors** | Curt statements; accusations on other issues; nonverbal anger; saying nothing |
| **Story Low Point** | Obsessed with perfection/rules/judgment leading to Fourish fantasy of micromanaging chaos |
| **Evolves To (safety)** | 7 -- Relax, appreciate beauty, let go of rules |
| **De-evolves To (fear)** | 4 -- Fantasy, micromanaging chaos |

## Type 2 -- The Helper

| Field | Data |
|-------|------|
| **Center** | Heart (Shame/Hostility) |
| **What We Like** | Real interest in others, good networkers, empathic, service oriented, genuinely caring |
| **What We Don't Like** | Needy, clingy, orchestrate others, hidden agendas, hostile, possessive |
| **Key Motivations** | Be loved, appreciated/needed, get positive responses |
| **Core Fear** | Being unlovable |
| **Poison** | Self-deception, prideful overvaluing of self |
| **Core Desire** | To be loved |
| **Distortion** | Insatiable need to be needed; controlling manipulation |
| **Focus of Attention** | Will others like me? Will I be needed? |
| **Survival Strategy** | Find the one with the power and make them love me |
| **Core Image** | I am loving, helpful |
| **Parental Matrix** | Ambivalent with father |
| **Speaking Style** | Ask questions, give compliments, focus on other person. Few self-references. Anger expressed indirectly. |
| **Speaking Attitude** | Soft-spoken, enticing, clingy, flirtatious, flattering, attentive |
| **Communication Blind Spots** | Secondary intention beneath generosity. Disengage quickly when uninterested. |
| **Distortion Filters** | Whether other likes them; whether they want to help; degree of other's influence |
| **Common Pinches** | Being taken for granted; feeling unappreciated; not being heard |
| **Pinch Behaviors** | Keep feelings to themselves long periods; intensely emotional when they speak; throw themselves on the fire |
| **Story Low Point** | Lash out or control person most important leading to Eightish domination: "I'll make you need me" |
| **Evolves To (safety)** | 4 -- Full emotional palette, creative impulses |
| **De-evolves To (fear)** | 8 -- Controlling, forceful, dominating |

## Type 3 -- The Achiever

| Field | Data |
|-------|------|
| **Center** | Heart (Shame/Hostility) |
| **What We Like** | Efficient, goal oriented, successful, quick, authentic, winners |
| **What We Don't Like** | Vain, phony, crave approval, phobic about failure, bad losers |
| **Key Motivations** | Affirmation, distinguish from others, crave attention, look good |
| **Core Fear** | Being valueless, not mattering |
| **Poison** | Self-delusion about superiority |
| **Core Desire** | To have value |
| **Distortion** | Chasing approval; resentment when withdrawn |
| **Focus of Attention** | How can I gain respect and esteem? |
| **Survival Strategy** | Get approval from as many as possible |
| **Core Image** | I am outstanding, a winner |
| **Parental Matrix** | Positive with mother |
| **Speaking Style** | Clear, efficient, logical, well-conceived. Quick on feet. Avoid topics where limitations show. |
| **Speaking Attitude** | Abrupt, dismissive, superficial, stony, high energy, slippery |
| **Communication Blind Spots** | Impatient with less capable; avoid discussing failings; appear driven; rush to decide |
| **Distortion Filters** | Whether info makes them look good; whether it interferes with goal; confidence of speaker |
| **Common Pinches** | Position of likely failure; not looking good; blamed for poor work; not receiving credit |
| **Pinch Behaviors** | Short structured questions; unlikely to say upset; sharp tone over time; clipped responses |
| **Story Low Point** | "I don't matter, nothing matters" leading to Nineish paralysis, lethargy, all achievement meaningless |
| **Evolves To (safety)** | 6 -- Achieve for others' safety, not approval |
| **De-evolves To (fear)** | 9 -- Lethargy, resentment, no value |

## Type 4 -- The Individualist

| Field | Data |
|-------|------|
| **Center** | Heart (Shame/Hostility) |
| **What We Like** | Sensitive, aesthetic, deep emotional connection, authentic, expressive, inquiring |
| **What We Don't Like** | Self-absorbed, self-referencing, melodramatic, needy, always longing |
| **Key Motivations** | Be special, express authentically, protect core emotions, find a rescuer |
| **Core Fear** | Innate ordinariness |
| **Poison** | Envy, self-absorption, moodiness |
| **Core Desire** | Sense of wholeness |
| **Distortion** | Neurotic self-absorption, searching for "the one" who can fix them |
| **Focus of Attention** | What is unavailable or missing? |
| **Survival Strategy** | Find someone who understands and can fix me |
| **Core Image** | I am unique, special |
| **Parental Matrix** | Negative with both parents |
| **Speaking Style** | Use I/me/my/mine frequently. Discuss feelings. Share personal/painful stories. Analytical introspection. |
| **Speaking Attitude** | Fluid, exaggerated, mercurial, emotionally charged, pensive, wistful, longing |
| **Communication Blind Spots** | Pull conversation to self; need to fully complete conversations; appear overly dramatic |
| **Distortion Filters** | Personal rejection; being slighted; not wanting to appear defective; being misunderstood |
| **Common Pinches** | Being ignored/slighted; asked to act against values; events eliciting envy |
| **Pinch Behaviors** | Blunt statements; become extremely quiet; multiple intense feelings; excessive analysis |
| **Story Low Point** | Overly involved, needy, desperate leading to Twoish neediness: obsessive focus on making others complete them |
| **Evolves To (safety)** | 1 -- Focus outward, self-discipline, create something special |
| **De-evolves To (fear)** | 2 -- Needy, desperate for connection |

## Type 5 -- The Investigator

| Field | Data |
|-------|------|
| **Center** | Head (Fear/Anxiety) |
| **What We Like** | Smart, analytic, systematic, fast decisions, cautious, visionaries |
| **What We Don't Like** | Withdrawn, paranoid, greedy with info, aloof, detached |
| **Key Motivations** | Possess knowledge, understand environment, defend emotional self with intellect |
| **Core Fear** | Incompetence, making mistakes |
| **Poison** | Panic, helplessness, compulsive wanting, loneliness |
| **Core Desire** | Appear competent always |
| **Distortion** | Useless specialization to feel accomplished |
| **Focus of Attention** | What data do I need to feel safe? |
| **Survival Strategy** | Trust my analysis, not my emotions |
| **Core Image** | I am perceptive, smart |
| **Parental Matrix** | Ambivalent toward both |
| **Speaking Style** | Speak tersely or in lengthy discourse. Highly selective word choices. Limited personal disclosure. |
| **Speaking Attitude** | Detached, aloof, self-contained, thorny, intellectual, distant, off-putting |
| **Communication Blind Spots** | May not show warmth; appear aloof/remote; say too much or too few words; appear withholding |
| **Distortion Filters** | Others' demands/expectations; feeling inadequate; intense emotions from others; invasion of privacy |
| **Common Pinches** | Breaking confidences; being surprised; dishonesty; out-of-control situations; overwhelm |
| **Pinch Behaviors** | Say little; pull back; retain experience mentally; may express anger as outrage later |
| **Story Low Point** | Thinking fails leading to Sevenish abandon, excess, numbness, unable to think or see clearly |
| **Evolves To (safety)** | 8 -- Visible, assertive, bring knowledge to world |
| **De-evolves To (fear)** | 7 -- Abandon thinking, numb with excess |

## Type 6 -- The Loyalist

| Field | Data |
|-------|------|
| **Center** | Head (Fear/Anxiety) |
| **What We Like** | Critical thinkers, proactive, loyal, courageous, troubleshooters, strategic |
| **What We Don't Like** | Catastrophize, worst-case, worry, doubt, frozen fear, distrustful of authority |
| **Key Motivations** | Safe and secure, feel supported, community and belonging |
| **Core Fear** | Being alone and unsupported |
| **Poison** | Reactive petulance, competitiveness, torpor |
| **Core Desire** | Safety and security |
| **Distortion** | Catastrophizing, planning paralysis |
| **Focus of Attention** | What might go wrong? |
| **Survival Strategy** | Anticipate all contingencies, prepare for worst case |
| **Core Image** | I am loyal, dependable |
| **Parental Matrix** | Positive with father |
| **Speaking Style** | Start with analytical comments. Alternate hesitant speech with bold/confident. Devil's advocate. |
| **Speaking Attitude** | Mentally energetic, hyper-vigilant, uptight, contrarian, diagnostic, on guard |
| **Communication Blind Spots** | Negative scenarios look like pessimism; self-doubt questions competence; worry appears as distrust |
| **Distortion Filters** | Whether authority is proper/improper; projection onto others; trust issues |
| **Common Pinches** | Pressure; lack of genuineness; lack of commitment; abusive authority |
| **Pinch Behaviors** | Withdraw; intensive analysis; highly reactive; project thoughts/feelings onto others |
| **Story Low Point** | Desperate to please then hostile toward fear sources leading to Threeish desperation for a win, any win |
| **Evolves To (safety)** | 9 -- Peace, relaxation, enjoy work |
| **De-evolves To (fear)** | 3 -- Desperate for a win, resentful |

## Type 7 -- The Enthusiast

| Field | Data |
|-------|------|
| **Center** | Head (Fear/Anxiety) |
| **What We Like** | Enthusiastic, fun-loving, lively, engaging, full of ideas, fast, glass-half-full |
| **What We Don't Like** | Addictive, obsessive, act before thinking, unfocused, scattered, judgmental |
| **Key Motivations** | Crave freedom/happiness, don't miss anything, stay distracted, avoid pain |
| **Core Fear** | Being consumed by pain |
| **Poison** | Gluttonous excess; critical perfectionism as escape |
| **Core Desire** | To be happy |
| **Distortion** | Frenetic escapism, addiction, distraction |
| **Focus of Attention** | What exciting possibility is next? |
| **Survival Strategy** | Shift attention to pleasant scenarios, run from pain |
| **Core Image** | I am fun, happy |
| **Parental Matrix** | Negative with mother |
| **Speaking Style** | Quick and spontaneous, words in a flurry. Engaging stories. Shift topic to topic. |
| **Speaking Attitude** | Rapid, light, unthreatening, upbeat, unpredictable, scattered, distracting |
| **Communication Blind Spots** | May not have absorbed all info they think; behavior undermines being taken seriously |
| **Distortion Filters** | Competence demeaned; think they know what other will say; having limits placed; being trapped |
| **Common Pinches** | Boring/mundane tasks; feeling dismissed; unjust criticism |
| **Pinch Behaviors** | Avoid by thinking of pleasant alternatives; rationalize own behavior; blame/condemn others |
| **Story Low Point** | Control excess by becoming perfect, constrained, righteous, blaming leading to Oneish judgmental rigidity |
| **Evolves To (safety)** | 5 -- Think deeply, savor solitude, slow down |
| **De-evolves To (fear)** | 1 -- Critical, righteous, perfectionist |

## Type 8 -- The Challenger

| Field | Data |
|-------|------|
| **Center** | Body (Anger/Resentment) |
| **What We Like** | Self-confident, decisive, protective of weak, leaders, champions, just, fair |
| **What We Don't Like** | Dictatorial, dismissive, willful, confrontational, punishing anger |
| **Key Motivations** | Self-reliant, not show weakness, be important, protect the defenseless |
| **Core Fear** | Own internal weakness and unreliability |
| **Poison** | Defensive confrontation, preoccupation with power/control, excess, morbid inaction |
| **Core Desire** | Prove self-reliance and strength |
| **Distortion** | Constant fighting and emotional combat to regain supremacy |
| **Focus of Attention** | Who has the power here? |
| **Survival Strategy** | Be strong, take control, show no weakness |
| **Core Image** | I am strong, just |
| **Parental Matrix** | Hate/love with both |
| **Speaking Style** | Statements to structure/control situation. Impatient with detail. Raise intensity to get truth. |
| **Speaking Attitude** | Intense, directive, bossy, terse, powerful, intimidating, in control, strategic, confrontational |
| **Communication Blind Spots** | Many are intimidated; energy stronger than they realize; vulnerability may show; seen as bully |
| **Distortion Filters** | Protecting those who need it; weakness in others; control issues; truthfulness; betrayal |
| **Common Pinches** | Injustice; not dealing directly; others not taking responsibility; being blindsided |
| **Pinch Behaviors** | Surges of anger; sift info quickly; avoid vulnerability; seek trusted counsel; direct confrontation |
| **Story Low Point** | Take total control leading to Fiveish paranoia, analysis-paralysis, reject everyone, go it alone |
| **Evolves To (safety)** | 2 -- Vulnerable, giving, helping others |
| **De-evolves To (fear)** | 5 -- Withdraw, paranoid, go it alone |

## Type 9 -- The Peacemaker

| Field | Data |
|-------|------|
| **Center** | Body (Anger/Resentment) |
| **What We Like** | Good listeners, flexible, compromising, accepting, find common ground, consensus builders |
| **What We Don't Like** | Self-abandoning, procrastinating, avoidant, passive-aggressive, distracted, merged |
| **Key Motivations** | Avoid conflict/tension, maintain homeostasis, find peace |
| **Core Fear** | Separation and disconnection |
| **Poison** | Lethargy, sloth, pathological self-abandonment |
| **Core Desire** | Peaceful, harmonious environment |
| **Distortion** | Give up more of self to avoid conflict; abandon self entirely |
| **Focus of Attention** | Will there be conflict? |
| **Survival Strategy** | Accommodate, merge, go along to get along |
| **Core Image** | I am easygoing, likable |
| **Parental Matrix** | Positive with both |
| **Speaking Style** | Highly detailed info in sequential style. Present all sides. May say yes but mean no. |
| **Speaking Attitude** | Deliberate, slow, non-confrontational, easygoing, incessant questioning, disengaged |
| **Communication Blind Spots** | Prolonged explanations lose listeners; multiple viewpoints reduce influence; fail to state position |
| **Distortion Filters** | Demands to change; being criticized/ignored; opposing views; anger possibly directed at them |
| **Common Pinches** | Disruption of peace; being told what to do; feeling ignored; rudeness; overt hostility |
| **Pinch Behaviors** | Say nothing; slight facial tension; may be unaware of own anger; displace anger onto safe targets |
| **Story Low Point** | Frozen in fear like deer in headlights leading to Sixish anxiety, worst-case paralysis, unable to act |
| **Evolves To (safety)** | 3 -- Personal presence, achievement, productive |
| **De-evolves To (fear)** | 6 -- Frozen in fear, worst-case paralysis |
